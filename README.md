# 🤖 AI Interview Assistant

An AI-powered Interview Preparation Web Application built using **Python (Flask), HTML, CSS, and JavaScript**.

## 🚀 Features

- 👤 Enter Your name
- 💻 Multiple Interview Domains
  - Python
  - Java
  - Web Development
  - HR Interview
- ⏳ 60-Second Timer
- 📊 Progress Bar
- 🎯 Score Calculation
- 💬 Instant Feedback
- 🌙 Dark Mode
- 📈 Final Result Card

## 🛠️ Technologies Used

- Python (Flask)
- HTML5
- CSS3
- JavaScript

## ▶️ How to Run

1. Clone the repository

```bash
git clone https://github.com/cutepieaditi123-wq/AI-Interview-Assistant.git
```

2. Install Flask

```bash
pip install flask
```

3. Run the application

```bash
python app.py
```

4. Open in browser

```
http://127.0.0.1:5000
```
